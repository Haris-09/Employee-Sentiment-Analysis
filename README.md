# Employee Sentiment Analysis

## 📌 Project Overview
This project analyzes employee messages to evaluate sentiment and engagement using NLP and statistical techniques.  
The workflow includes:
- Sentiment labeling
- Exploratory Data Analysis (EDA)
- Employee score calculation
- Employee ranking
- Flight risk identification
- Predictive modeling

---

## 🚀 How to Run
1. Clone the repository.
2. Place `test.csv` in the root directory.
3. Open `Employee_Sentiment_Analysis.ipynb` in Jupyter Notebook.
4. Run all cells sequentially.

Dependencies:
- Python 3.8+
- PyTorch / scikit-learn
- HuggingFace Transformers
- Pandas, NumPy, Matplotlib, Seaborn
